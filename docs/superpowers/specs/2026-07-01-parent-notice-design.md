# STU-PAR-01 家长知情页面设计

## 目标

把 `/stu/parent-notice` 从占位页升级为可验收的学生端页面，并在现有 Figma 文件中建立对应的 `STU-PAR-01：家长知情` Frame。

## 页面结构

- 复用学生端 1440px 桌面外壳：232px 深蓝侧栏、72px 顶栏、浅灰内容背景。
- 页面标题区展示“家长知情”和“查看告知书并提交家长签字回执”。
- 顶部状态卡展示当前实习、确认状态、回执截止日期。
- 主体左栏展示学生、实习单位、岗位、周期及家长联系方式。
- 主体右栏展示家长告知书摘要，提供“预览告知书”和“下载告知书”。
- 底部回执卡展示文件名、上传时间、审核状态，并提供重新上传入口。

## 状态与交互

- 默认演示态为“审核中”，与申请表已有“家长知情回执”材料项衔接。
- “预览告知书”展开页内预览区域；“下载告知书”使用浏览器下载模拟。
- “重新上传”触发隐藏文件选择器；选择 PDF/JPG/PNG 后更新文件名和状态提示。
- 页面不增加在线验证码确认，避免与当前回执上传流程产生两套口径。

## 验收标准

- 路由和侧边栏入口保持不变。
- 页面不再包含占位文案。
- 关键按钮有明确演示行为。
- 1440px 桌面布局无明显溢出、遮挡或空白断层。
- Figma Frame 与浏览器页面主要结构、文案、颜色和间距一致。
