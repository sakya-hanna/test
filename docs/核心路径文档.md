## 验收标准

每条路径验收时检查：
- [ ] 每一步的页面可渲染（200 OK，无控制台红错）
- [ ] 每一跳的入口按钮存在且可触发
- [ ] 每一步有返回上一级/来源页面的按钮
- [ ] 流程可正向走完 → 反向返回（A→B→C→B→A）
- [ ] 面包屑与实际页面层级一致
